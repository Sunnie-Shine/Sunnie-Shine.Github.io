有没有想过，它这么书写就非常像是高级编程语言里的对象实例化和对象调用了。

我定义了一个 Object 类型的对象，这个对象啥都没有，没有字段，没有所谓的属性和其它东西，但它却具有一个函数指针。这个函数指针在 createInstance 方法下被初始化，指向 print 函数。当我们需要使用的时候，使用星号进行解取地址操作，然后调用方法，传入参数 32。

注意，函数指针在前文里说过，一般它的使用就是指向函数和控制和间接调用函数，所以它可以不写星号也可以成功，故这个代码 (*o->print)(32) 也可以不要前面的括号和星号：o->print(32)。
