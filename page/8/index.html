词牌和规则

抽空列举一些我之前收集到的词牌名和一些规则。

忆王孙

单调三十一字，五平韵，句句用韵，亦有将单片重复做双调者。
词牌格式：
(平)平(仄)仄仄平平，(仄)仄平平(仄)仄平。
(仄)仄平平(仄)仄平。仄平平。(仄)仄平平(仄)仄平。
李重元：
萋萋芳草忆王孙，柳外楼高空断魂。
杜宇声声不忍闻。欲黄昏。雨打梨花深闭门。

如梦令

单调三十三字，五仄韵，一叠韵，上去通押。
词牌格式：
(仄)仄(仄)平平仄，(仄)仄(仄)平平仄。
(仄)仄仄平平，(仄)仄(仄)平平仄。平仄，平仄(叠句)，(仄)仄(仄)平平仄。
李清照：
常记溪亭日暮，沉醉不知归路。兴尽晚回舟，误入藕花深处。争渡，争渡，惊起一滩鸥鹭。

乌夜啼

双调三十六字，前阕三平韵，后阕两仄韵、两平韵。
词牌格式：
(平)平(平)仄平平，仄平平。(仄)仄(平)平平仄、仄平平。
(仄)(平)仄，(仄)(平)仄，仄平平。(仄)仄(平)平平仄、仄平平。
李煜：
无言独上西楼，月如钩。寂寞梧桐深院锁清秋。
剪不断，理还乱，是离愁。别是一般滋味、在心头。

长相思

双调三十六字，前后阕格式相同，各三平韵，一叠韵，一韵到底。
词牌格式：
仄(仄)平，仄(仄)平(叠后二字)，(仄)仄平平(仄)仄平。(平)平(仄)仄平。
仄(仄)平，仄(仄)平(叠后二字)，(仄)仄平平(仄)仄平。(平)平(仄)仄平。

思悠悠，恨悠悠，恨到归时方始休。明月人倚楼。

生查子

词牌格式：
(平)平(仄)仄平，(仄)仄平平仄。(仄)仄仄平平，(仄)仄平平仄。
(平)平(仄)仄平，(仄)仄平平仄。(仄)仄仄平平，(仄)仄平平仄。

今年元夜时，月与灯依旧。不见去年人，泪满春衫袖！

点绛唇

双调，上片四句，押三仄韵，二十字，下片五句，押四仄韵，二十一字，用仙吕调，共四十一字。

(仄)仄平平，(仄)仄平平仄。平平仄，(仄)平(平)仄，(平)仄平平仄。

来是春初，去是春将老。长亭道，一般芳草，只有归时好。

is 运算符

关键字介绍

is 就好比 Java 语言里的 instanceof，在运算符左侧写的是变量，右侧写的是数据的类型。它会尝试测试和判定变量是否是这个类型的。如果是，则返回 true；而否则返回 false，如下示例。

可以看到的是，只要兼容的类型，它都能测试出来。例如，字符串实现了 IEnumerable<char> 接口，那么它就完全可以转换为这个类型。在 OOP 的世界里，一个类型就好比是一个角色。你实现（或继承）了什么类型，就拥有了什么角色，你就可以转换过去。

一切皆对象（object），那么我们所有人，所有事都可以被转换为 object 类型。包括值类型，它会迂回地转换，先隐式转换（派生自）ValueType 类类型，而这个类型又是派生自 object 的；而指针，则隐式继承自 IntPtr 结构类型，而这个类型依然派生自 object。
